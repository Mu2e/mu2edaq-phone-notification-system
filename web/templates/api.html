ZeroMQ compatibility

When zmq.enabled is true the server also accepts events on ZeroMQ: SUB sockets connected to the endpoints in zmq.connect and a PULL socket bound at zmq.bind. JSON payloads use the same schema as POST /api/events; plain-text payloads become events with the configured default severity.
